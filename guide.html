Hướng dẫn

  1. Tải bộ cài phù hợp từ trang Tải game.
  2. Giải nén (nếu cần) và chạy trình cài đặt.
  3. Mở game, đăng ký tài khoản trong mục Đăng ký.
  4. Nếu lỗi mở game, cài thêm Microsoft Visual C++ Redistributable.

Tools

  • Auto Update: tự động kiểm tra bản vá.
  • Repair: quét và sửa file hỏng.
