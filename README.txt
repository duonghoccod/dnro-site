Dnro-like Site (Demo)
----------------------
Thư mục này chứa mã nguồn website tĩnh, đa trang, tách riêng HTML/CSS/JS.

Cấu trúc:
- index.html, download.html, news.html, guide.html, faq.html, play.html
- assets/css/reset.css, assets/css/styles.css
- assets/js/main.js, news.js, launcher.js, news.json
- assets/img/logo.png, hero.jpg, favicon.png

Triển khai:
- Dùng bất kỳ web server tĩnh nào (Nginx, Apache, Vercel, GitHub Pages).
- Hoặc chạy Python: `python3 -m http.server` trong thư mục này.

Lưu ý pháp lý:
- Đây là mẫu gốc, KHÔNG sao chép nội dung/nhãn hiệu của trang khác.
- Hãy thay thế hình ảnh, tên dự án, domain theo nhu cầu của bạn.
